# 程序员爱分享

该小程序功能比较简单，微信用户每天签到送积分，积分可以兑换视频资源

### 前端

微信小程序前端框架使用ivew weapp

### 后端

后端框架使用JFinal(国产框架)

### 数据库

数据库使用Mysql
